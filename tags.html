Um pouco sobre Tags

Tag para Título

< h1> a < h6>: Definem títulos ou cabeçalhos, com < h1> sendo o maior e < h6> o menor. Existe uma forma hierárquica para títulos, indicando a estrutura do conteúdo.

Tag para Formatação de texto

São existem diferentes formas de formatar um texto, essa formatação vai depender de como você vamos ver cada uma delas.

Tag de Lista

São existem diferentes formas de declarar uma lista, vamos ver cada uma delas.

Lista ordenadas: Contêm vários itens numerados.

Listas não ordenadas: com vários itens marcados todos com o mesmo símbolo.

Listas de definições: é constituida por uma série de itens mas por termos acompanhados das descrições de seus significados.

Tag para Link

As tags de links são fundamentais para a criação de páginas web interativas. A tag < a > é usada para criar links em uma página HTML. O atributo href especifica o destino do link.

A tag < img > é usada para incorporar imagens em uma página HTML. O atributo src especifica o caminho da imagem, e o atributo alt fornece um texto alternativo para acessibilidade.
